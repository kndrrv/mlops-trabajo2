Proyecto MLOps - Predicción de Stroke
Estudiante: Kendra Gutiérrez Vega
Dataset: Stroke Prediction Dataset
Descripción
Pipeline de Machine Learning para predecir stroke usando Random Forest.

Archivos:
load_data.py - Carga de datos
preprocessing.py - Preprocesamiento
training.py - Entrenamiento del modelo
evaluation.py - Evaluación del modelo

Cómo usar:
# 1. Cargar datos
python load_data.py

# 2. Preprocesamiento
python preprocessing.py

# 3. Entrenar modelo
python training.py

# 4. Evaluar
python evaluation.py

Dataset
Descargar de: https://www.kaggle.com/datasets/aouatifcherdid/healthcare-dataset-stroke-data
